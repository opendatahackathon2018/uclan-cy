What is this project about?

This project ios about collecting and maintaining important data related to water resources in Cyprus and making them available as Open Data. We also built a fun quiz that aims to help teenagers understand volume quantities. Try it if you want to find out how many elephants fit in an olympic pool.

Can I see the data and code?

Sure you can. The data and the code are all available on the cloud. You can find the MIT License code on GitHub at https://github.com/nearchos/cyprus-water. You can also find the API docs at http://cyprus-water.appspot.com.

Who developed this?

We are a team of open data and open source code enthusiasts. We built this project as part of #opendatacy hackathon 2018.
